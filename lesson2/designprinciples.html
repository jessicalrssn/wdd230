Design Principles

Jessica Larsson

White Space and Clean Design

Chick-fil-a

Chick-fil-a.com

This website has a lot of white space and a very clean desig. it is easy to see the important things and all the elements jump out because of the use of white space.

Contrast

National Geographic

nationalgeographic.com

This website has very good contrast throughout. the white letters on the black background really make it stand out.

Repetition

The Grand America Hotel

grandamerica.com

You can't see it from this screenshot but this website uses a lot of repetition. It uses the same format for every section you come across while scrolling. The alignment, colors, picture placement, etc. all repeat through the whole site.
